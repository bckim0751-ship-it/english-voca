# 영단어 마스터 📚

매일 10개의 영단어를 학습할 수 있는 웹 기반 영어 단어 학습 시스템입니다.

## 기능

### 오늘의 단어
- 날짜 기반으로 매일 새로운 10개 단어 자동 선택
- 플래시카드 방식으로 단어 학습 (탭하여 뜻 확인)
- 각 단어에 대해 "알았어요" / "다시 보기" 표시
- 학습 진행률 실시간 확인

### 퀴즈
- 오늘 학습한 10개 단어로 퀴즈 진행
- 영어→한국어, 한국어→영어 교차 출제
- 4지선다 객관식
- 퀴즈 완료 후 정답/오답 상세 확인

### 단어장
- 300개 전체 단어 목록 확인
- 기초 / 중급 / 고급 레벨별 필터링
- 각 단어의 뜻, 품사, 예문 제공

### 통계
- 총 학습일 및 학습 단어 수
- 연속 학습 스트릭
- 전체 퀴즈 정답률
- 최근 퀴즈 기록

## 단어 구성

| 레벨 | 단어 수 | 수준 |
|------|---------|------|
| 기초 | 100개 | 일상 기본 어휘 (CEFR A2-B1) |
| 중급 | 100개 | 업무/학문 어휘 (CEFR B2) |
| 고급 | 100개 | 학술/전문 어휘 (CEFR C1-C2) |

## 사용 방법

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.
별도의 설치나 서버가 필요하지 않습니다.
학습 기록은 브라우저의 localStorage에 저장됩니다.

## 파일 구조

```
├── index.html
├── css/
│   └── style.css
└── js/
    ├── words.js    # 300개 단어 데이터
    └── app.js      # 앱 로직
```
